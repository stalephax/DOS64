  HOW TO BUILD THAT TING?

  First, you need to get a UNIX-like build environnement (you can either use WSL, or a Linux OS if you have it installed on your PC, or Cygwin)
  Make sure that MAKE is available as a command (type make ? , and if it throws "make: *** No rule to make target '?'.  Stop." means that it is installed :) )
  Then you have just to type make clean (clean build and nuke the old binaries) and then "make" alone without quotes, to generate the ISO.
